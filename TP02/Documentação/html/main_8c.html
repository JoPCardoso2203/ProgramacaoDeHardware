Este Tp é o segundo TP da disciplica de Programãção de Hardware.

O aluno(a) deverá desenvolver um código em C, em que ele deverá criar as funções de manipulação de IO (usando os registros DDRx,PINx,PORTx) onde cada deverá proporcionar ao usuário poder escolher qualquer pino e port que ele quiser. Em seguida, na função main, o aluno deverá implementar um exemplo usando as funçoes, e de preferencia para implementar como se fosse um projeto (de sua escolha. EX: Sensor de janelas).A resposta deverá ser implementado cabeçalho de acordo com o exemplo deste e descrição de cada comando usando Doxygen. A resposta deverá ser enviada para o próprio github do aluno, e postar os arquivos fonte na tarefa do teams, deve-se postar os arquivos em si(não postarem em formato zip).
